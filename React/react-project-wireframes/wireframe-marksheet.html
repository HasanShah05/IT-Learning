Project 1: Marksheet Generator

A school manages students, enters marks for 4 exam types, and generates marksheets with grades

Project Overview: This app has 4 types of exams — Unit 1 Test (max 30), Unit 2 Test (max 30), Term 1 Exam (max 80), Term 2 Exam (max 80). A teacher adds students, enters marks per exam, and can view a marksheet for any student for any exam, or a consolidated report across all 4 exams.
